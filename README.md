## 🌐 Tentang Proyek
Ini adalah repositori resmi untuk website portofolio pribadi **Muslim Hacker Division**. Sebuah platform digital modern bergaya *cyberpunk-terminal interface* yang dirancang untuk merepresentasikan harmoni sempurna antara empat pilar keahlian utama: **Agama, Teknologi, Seni Ilusi (Sulap), dan Strategi Peperangan**.

Website ini sudah dioptimasi agar responsif, cepat, dan menggunakan standar pemisahan kode (*Clean Code Architecture*) untuk kemudahan perawatan jangka panjang.

---

## 🛠️ Tech Stack & Fitur Utama

### **Teknologi yang Digunakan:**
* **Frontend:** HTML5, CSS3 (Custom Variables & Grid System), Vanilla JavaScript (ES6+).
* **Fonts & Icons:** Google Fonts (Orbitron & Rajdhani), Font Awesome v6.4.0.
* **Hosting:** GitHub Pages.

### **Fitur Repo / Desain Modern:**
* 🌟 **Modular Code Structure:** Pemisahan kode yang bersih antara struktur (`index.html`), desain (`style.css`), dan logika (`script.js`).
* 💻 **Interactive Terminal UI:** Komponen simulasi terminal bash interaktif yang memberikan aura teknis yang kuat.
* 📱 **Responsive Design:** Optimalisasi penuh untuk tampilan layar ponsel cerdas (Mobile) maupun perangkat desktop.
* 🛎️ **Toast Notification System:** Fitur umpan balik instan berbasis JavaScript saat pengguna berinteraksi dengan elemen web.
